# RentForge

Construction Equipment Rental Management System built using:

- React.js
- Spring Boot
- MySQL
- Firebase Authentication
- Razorpay Payment Gateway

## Features

- User Authentication
- Equipment Management
- Multi-unit Inventory
- Date-wise Booking
- Booking History
- Admin Dashboard
- Online & Cash Payment
- Security Deposit System

## Tech Stack

Frontend:
- React
- Tailwind CSS
- Axios

Backend:
- Spring Boot
- JPA/Hibernate
- MySQL

Authentication:
- Firebase Google Login

Payment:
- Razorpay
